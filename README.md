# ConversorDec2Bin
 
## Utilidad
 
  #### Este programa se encarga de pasar números tanto positivos como negativos en sistema decimal a binario.
  
## Uso

  1. Abre el archivo dec2bin.py para inicoar el programa.
  2. Escriba primero el número en decimal que quiera pasar a binario
  3. Escriba la cantidad de dígitos con la que quiere que se le enseñe el número en binario. En caso de que falten el programa los añadirá automaticamente.
  4. Al final se le mostrará el número deseado en sistema binario.
